⸻

PRODUCT NAME

One-Prompt AI App Generator

⸻

CORE PURPOSE

Build an AI system that accepts one single prompt from a user and outputs a complete, production-ready, monetised web application, including:
	•	Frontend
	•	Backend
	•	AI integration
	•	Payments

The generated apps must be ready to deploy immediately on any modern hosting platform.

	•	Do NOT require interactive setup
	•	Do NOT require the generator to host the generated apps
	•	Do NOT require third-party automation tools
	•	Everything must be automated with sensible defaults

⸻

GENERATOR INPUT
	•	A single large text prompt describing the desired app idea

⸻

GENERATOR OUTPUT (CRITICAL)

For each prompt, output:
	•	A complete full-stack web app (frontend + backend)
	•	OpenAI integration using /v1/responses
	•	Stripe payments fully wired
	•	Clear environment variable list
	•	Deployment-ready project structure

Do NOT deploy the generated app.
The generator’s job is to produce deployable, monetised apps — not to host them.

⸻

AI / BACKEND REQUIREMENTS (GENERATED APPS)

	•	Enforce strict token limits
	•	Return clean plain-text output by default

MONETISATION REQUIREMENTS (GENERATED APPS)

Every generated app must include built-in monetisation with the following properties:

Supported Monetisation Modes
	•	One-time payment (unlock unlimited usage)
	•	Subscription (monthly recurring access)

IMPORTANT

	•	Instead, the generated app itself must include an owner-only Monetisation Settings UI

	•	One-time payment
	•	Subscription

Defaults
	•	Default monetisation mode: one-time payment
	•	Include sensible default pricing
